Frameset adalah tag HTML lama yang digunakan untuk membagi jendela browser menjadi beberapa bagian yang disebut frame. Setiap frame dapat menampilkan file HTML yang berbeda, sehingga memungkinkan adanya menu di satu sisi dan konten di sisi lain dalam satu halaman. Frameset biasanya menggunakan atribut cols untuk membagi layar secara vertikal atau rows untuk membagi secara horizontal, sedangkan frame berfungsi menampilkan file HTML di dalam area tersebut.

Setiap frame dapat diberi atribut seperti src untuk menentukan halaman default, name untuk memberikan identitas frame, serta scrolling, noresize, dan border untuk mengatur tampilan maupun perilakunya. Interaksi antar frame terjadi melalui kombinasi name dan target, misalnya sebuah link di frame menu dapat diarahkan untuk membuka halaman baru di frame konten dengan target Isi.

Walaupun dulu sangat populer untuk membuat layout sederhana dengan header, footer, sidebar, dan content, penggunaan frameset kini sudah tidak direkomendasikan. Kelemahannya antara lain sulit dikendalikan dengan CSS modern, tidak ramah SEO, dan menyulitkan aksesibilitas. Saat ini, layout web lebih sering dibuat dengan CSS seperti Flexbox atau Grid, sedangkan kebutuhan untuk menampilkan konten dari luar dapat menggunakan iframe.
